Abstract sound of the Amazons

Este aplicativo web se publicó en un repositorio en Git Hub para que distintos usuarios tuvieran la posibilidad de interactuar desde sus ordenadores.

El código base se realizó en Html (lenguaje de marcado de hipertexto) utilizando adicionalmente la API (Interfaz de programación de aplicaciones) de Audio de JavaScript (lenguaje de programación interpretado).

Se seleccionaron archivos de audio de uso libre para elaborar esta pieza sonora y se editaron en el programa Reaper (estación de trabajo de audio digital).
